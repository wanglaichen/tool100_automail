四、MailSlurp的token：
  在 Vercel 上这样设置：

  1. 打开 Vercel Dashboard
  2. 进入你的项目
  3. Settings → Environment Variables
  4. 新增变量：

  Name:  MAILSLURP_API_KEY
  Value: 你的 MailSlurp API Key
  Environment: Production

  5. 保存后重新部署一次
